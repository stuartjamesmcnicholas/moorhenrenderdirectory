# Multiple file loading and rendering using Moorhen's libCootApi

The intention is to build a program that one can load a whole bunch of files at once
into and then render an image of the molecule. Currently, this is in development -
the files are loaded and glTF meshes produced, just for testing prurposes.

To use:

* `npm install`
* Copy `moorhen64.js` and `moorhen64.wasm` (or `moorhen.js` and `moorhen.wasm`)
from a moorhen installation into `src`.
* `npm run dev`
